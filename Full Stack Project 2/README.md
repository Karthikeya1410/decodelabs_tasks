# Full Stack Project 2

## Backend API Development

This project follows the PDF requirement: create a simple backend API with GET and POST endpoints, user input handling, responses, and basic validation.

## API endpoints

- `GET /api/interns` - returns the intern list
- `POST /api/interns` - creates an intern

## How to run

```bash
node server.js
```

Open `http://localhost:3000/api/interns` in a browser to test the GET endpoint.
