# Full Stack Project 4

## Frontend and Backend Integration

This project follows the PDF requirement: connect the frontend and backend using HTTP requests, REST-style endpoints, `fetch`, and `async/await`.

## Features

- Frontend form submits intern data
- Backend receives POST requests
- Frontend loads data using GET requests
- Data is saved in `database.json`

## How to run

```bash
node server.js
```

Open `http://localhost:3000` in a browser.
