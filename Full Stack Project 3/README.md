# Full Stack Project 3

## Database Integration

This project follows the PDF requirement: connect a backend with a database, design a simple schema, perform CRUD operations, and handle data properly.

## API endpoints

- `GET /api/interns` - read all interns
- `POST /api/interns` - create an intern
- `PUT /api/interns/:id` - update an intern
- `DELETE /api/interns/:id` - delete an intern

## How to run

```bash
node server.js
```

The data is stored in `database.json`.
